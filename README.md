# FuckYourACE

一个基于 Wails 的 Windows 桌面工具，旨在降低特定后台进程（如 SGuard64.exe, SGuardSvc64.exe）的系统资源占用。

## 功能

本工具会执行以下操作：

1.  **查找目标进程**：自动搜索系统中正在运行的 `SGuard64.exe` 和 `SGuardSvc64.exe` 进程。
2.  **绑定核心**：将找到的目标进程强制绑定到 CPU 的第一个“能效核”（E-Core）。
    * 如果系统没有能效核（E-Core），它将作为备用方案，绑定到最后一个可用的逻辑核心。
3.  **降低优先级**：将这些进程的 CPU 优先级设置为“最低”（Idle）。

程序启动后，上述流程会**每隔 60 秒**自动循环执行一次，以确保设置持续生效。

## 为什么这么做？

某些后台服务（尤其是游戏的反作弊程序）可能会持续占用大量 CPU 资源，导致游戏或其他应用程序的性能下降。

通过将这些进程“锁”在单个能效核上并将其优先级降至最低，可以显著减少它们对系统性能的影响，同时不终止它们（这可能会导致游戏关闭）。

## 如何构建

这是一个 Wails v2 项目。

1.  确保你已安装 [Wails v2 环境](https://wails.io/docs/gettingstarted/installation)。
2.  克隆本仓库。
3.  进入 `frontend` 目录并运行 `npm install` 来安装前端依赖。
4.  返回项目根目录，运行 `wails build` 来构建可执行文件。
